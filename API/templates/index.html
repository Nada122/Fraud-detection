API Access

The system also provides API endpoints for programmatic access:

  • GET /api/fraud_data - Get all records
  • POST /api/fraud_data - Add a new record
  • GET /api/fraud_data/{index} - Get a specific record
  • PUT /api/fraud_data/{index} - Update a record
  • DELETE /api/fraud_data/{index} - Delete a record
  • GET /api/fraud_data/account/{account_id} - Get records by account ID
